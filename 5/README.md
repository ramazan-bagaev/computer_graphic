# Кодирование и декодирование по методу Хоффмана

## Установка

```
mvn install
```

## Использование

### Кодирование

```
java -jar target/task5-1.0.jar encode #имя файла
```

результат работы файл encoded



### Декодирование

```
java -jar target/task5-1.0.jar decode #имя файла
```

результат работы файл decoded
